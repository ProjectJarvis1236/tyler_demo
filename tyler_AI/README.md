Tyler — это AI-помощник, который помогает управлять проектами и автоматизировать повседневные задачи. 
В этом репозитории представлена backend-часть проекта. Для публикации были удалены конфиденциальные данные.


Основные фреймворки: FastAPI, Uvicorn, Pydantic, Sentence Transformers, ONNX Runtime, Qdrant, Vosk, OpenPyXL, Pandas, python-docx, pdfplumber, Watchdog, AsyncIO


В рамках разработки backend-части Tyler мной реализованы:
1. Оркестрация выполнения задач пользователя и взаимодействия с LLM.
2. Система действий для управления приложениями, окнами и браузером.
3. Работа (анализ, создание, редактирование) с документами Microsoft Word, Excel и прочими.
4. Индексация файлов и интеграция с системой памяти. Заложены основы семантического поиска.
5. Веб-поиск и получение актуального контекста для LLM.
6. Голосовой режим с распознаванием речи и поддержкой wake word.
7. Реализация компонентов долговременной памяти и вспомогательных сервисов.
